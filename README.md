# The Nova Edits

A modern, responsive web application showcasing products with features including a shopping cart, new arrivals section, and custom styling.

## 🌐 Live Demo

**Check out the live website here:** [**The Nova Edits - Live Demo**](https://thenovaedits-70544.web.app)

## 📋 Overview

The Nova Edits is a sleek frontend project designed to provide an engaging user experience for browsing and purchasing products. The site includes multiple pages, dynamic shopping cart functionality, and a professional visual design.

## 🚀 Features

- **Responsive Design** - Mobile-friendly and works across all devices
- **Shopping Cart** - Add and manage products in your cart
- **New Arrivals** - Dedicated section for latest products
- **Custom Styling** - Professional CSS styling with consistent branding
- **Dynamic Interactions** - JavaScript-powered interactive elements
- **Error Handling** - Custom 404 error page

## 📖 Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/devrizu/the-nova-edits.git
   ```

2. Navigate to the project directory:
   ```bash
   cd the-nova-edits
   ```

3. Open `index.html` in your web browser or use a local server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (if http-server is installed)
   http-server
   ```

4. Browse the site and explore all pages including the shopping cart functionality.

## 🎨 Pages

- **Home** (`index.html`) - Main landing page
- **New Arrivals** (`newarrivals.html`) - Latest products
- **Shopping Cart** (`shopping-cart.html`) - Cart management
- **404** (`404.html`) - Error page for missing routes
